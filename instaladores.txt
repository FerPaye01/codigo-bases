================================================================================
GUÍA DE INSTALACIÓN Y LISTA DE DEPENDENCIAS - ASISTENTE DE BASES ESTÁNDAR
================================================================================

Este archivo detalla todas las herramientas y dependencias necesarias para ejecutar 
el Asistente de Bases Estándar localmente, desde los servicios del LLM hasta las 
librerías de Python.

--------------------------------------------------------------------------------
1. MOTOR DE INFERENCIA LLM LOCAL: OLLAMA
--------------------------------------------------------------------------------
El proyecto requiere ejecutar Ollama localmente en la máquina del usuario.

* Descarga oficial: https://ollama.com/download (Soporta Linux, macOS y Windows).
* Instalación en Linux (por comando):
  curl -fsSL https://ollama.com/install.sh | sh
* Inicio del servicio:
  ollama serve
* Descarga del modelo configurado (Gemma 3 de 1B de parámetros):
  ollama pull gemma3:1b

--------------------------------------------------------------------------------
2. BASES DE DATOS LOCALES (SIN COSTO DE LICENCIA)
--------------------------------------------------------------------------------
* Base de Datos Transaccional (SQLite):
  Incluido nativamente en la instalación estándar de Python. No requiere servidores 
  externos ni instaladores adicionales. El esquema se genera e inicializa automáticamente 
  en el arranque de la app.
* Base de Datos Vectorial (Qdrant):
  Se ejecuta en modo local-serverless embebido mediante el cliente `qdrant-client` y 
  `fastembed`, usando almacenamiento directo en disco. No requiere levantar contenedores 
  Docker ni configurar servicios adicionales.
